# Interpolation de Lagrange

Cette application web permet aux utilisateurs de saisir plusieurs points et effectue une interpolation de Lagrange sur ces points, affichant la fonction polynomiale résultante.

## Fonctionnalités
- Interface utilisateur conviviale pour saisir les points
- Effectue automatiquement l'interpolation de Lagrange sur les points saisis
- Affiche la fonction polynomiale résultante
- Possibilité de copier la fonction résultante (LaTeX)

## Essayer l'application
Vous pouvez essayer l'application d'interpolation de Lagrange en [cliquant ici](https://joselor2004.github.io/InterLagrange/)
